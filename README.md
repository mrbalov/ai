# AI Skills & Tools

This repository contains reusable AI skills and automation tools for AI.

## 🛠️ Skills

| Skill | Description | Documentation |
|-------|-------------|-----------------|
| **document-ai** | Discovers all available AI tools (skills, hooks, and agents); generates comprehensive AI.md with AI tool inventory, decision trees, and an Agent Tools Graph showing automation infrastructure. Catalogs AI capabilities from: .kiro/skills/, .claude/skills/, skills-lock.json, and enabled plugins. Also generates a Mermaid diagram visualizing hook→agent→skill→document relationships and automation workflows. Includes document/file dependency discovery — every file referenced by an AI entity appears in the graph with usage context. | [SKILL.md](skills/document-ai/SKILL.md) |
